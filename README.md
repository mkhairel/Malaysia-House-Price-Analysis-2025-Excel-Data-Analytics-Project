# Malaysia-House-Price-Analysis-2025-Excel-Data-Analytics-Project
Analisis faktor yang mempengaruhi harga rumah di Malaysia menggunakan Excel dan membina dashboard interaktif untuk insight pasaran.

Business Questions?
1. Negeri mana mempunyai harga rumah paling tinggi?
2. Apakah faktor paling mempengaruhi harga? (sqft, bilik, lokasi)
3. Bagaimana harga berbeza mengikut jenis rumah
4. Adakah terdapat korelasi antara keluasan dan harga?
5. Apakah anggaran harga berdasarkan keluasan tertentu?

Kwmahirann digunakan dalam analisis:
1. Data Cleaning
2. Exploratory Data Analysis
3. Advanced Analysis
4. Dashboad
